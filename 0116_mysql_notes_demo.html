___#0116 MySQL notes
-------------------------- CRUD operations
| C -create, add or insert
| R -read
| U -update
| D -delete


| _____DBMS <> database mgmt system
----------------used to interpret SQL instructions.


| __#0116 data sub languages:
| _____DML <> data manipulation language
| CRUD operations fall here.
| insert > to add data to a table
| update > edit data already in table
| delete > remove data from table

| _____DQL <> data query language
| select cmd used to get data from 1+ tables

| _____DCL <> data control language
| control access to db

| _____DDL <> data definition language
| create > create a table
| alter > modify structure of table
| drop > remove an object like a table from a database
